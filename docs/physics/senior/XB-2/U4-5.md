## 概说

无线电波与移动互联网是高中物理选择性必修二第四章的应用部分。本节主要研究移动通信的基本原理、电磁波频段的实际应用以及移动互联网的发展。

## 移动通信的基本原理

### 蜂窝移动通信系统

现代移动通信采用**蜂窝网络**结构，将服务区域划分为若干个小区，每个小区设置一个基站，形成近似六边形的覆盖区域。

!!! info "蜂窝结构优点"
    1. 频率复用：相邻小区使用不同频率，非相邻小区可使用相同频率，提高频谱利用率
    2. 覆盖均匀：各小区覆盖范围相近，用户分布均匀
    3. 功率控制：移动台根据与基站的距离自动调整发射功率，降低干扰

### 移动通信的工作过程

```
移动台 → 基站 → 移动交换中心 → 公共电话网/互联网
```

1. **上行链路**：移动台（手机）将语音或数据信号调制后，通过无线电波发送到基站
2. **基站处理**：基站接收信号，进行放大、解调、复用等处理
3. **网络传输**：通过光纤或微波链路传输到移动交换中心
4. **下行链路**：信号经过处理后，通过另一个基站发送到目标移动台

### 越区切换

当移动用户在通话过程中穿越小区边界时，系统自动将通信从一个基站切换到另一个基站，保持通话连续。

!!! note "切换策略"
    - 硬切换：先断开原有连接，再建立新连接
    - 软切换：同时与两个基站保持连接，再断开其中一个

## 电磁波频段的应用

### 移动通信频段

我国主要使用的移动通信频段：

| 频段 | 频率范围 | 应用 |
|:----:|:--------:|:----:|
| 800MHz频段 | $825\text{MHz} \sim 835\text{MHz}$ / $870\text{MHz} \sim 880\text{MHz}$ | 早期CDMA |
| 900MHz频段 | $890\text{MHz} \sim 915\text{MHz}$ / $935\text{MHz} \sim 960\text{MHz}$ | GSM（2G） |
| 1800MHz频段 | $1710\text{MHz} \sim 1785\text{MHz}$ / $1805\text{MHz} \sim 1880\text{MHz}$ | DCS（2G） |
| 2100MHz频段 | $1920\text{MHz} \sim 1980\text{MHz}$ / $2110\text{MHz} \sim 2170\text{MHz}$ | WCDMA（3G） |
| 2600MHz频段 | $2500\text{MHz} \sim 2690\text{MHz}$ | LTE（4G） |
| 3300MHz~3600MHz | - | 5G（Sub-6G） |
| 毫米波 | $24GHz \sim 52GHz$ | 5G毫米波 |

### 频段与传播特性

- **低频段**（如800MHz、900MHz）：波长较长，穿透能力强，覆盖范围广，适合农村和室内覆盖
- **高频段**（如2600MHz、毫米波）：波长较短，容量大，但穿透能力弱，覆盖范围小，适合城市热点区域

!!! tip "实际应用"
    运营商通常采用"低频+高频"组合的方式，低频保证覆盖，高频提供容量。

## 4G与5G技术

### 4G（LTE）

**长期演进**（LTE，Long Term Evolution）是第四代移动通信技术，主要特点：

- **高速率**：峰值速率可达 $100\text{Mbps}$（下行），$50\text{Mbps}$（上行）
- **低时延**：端到端时延约 $20\text{ms} \sim 50\text{ms}$
- **宽带接入**：支持移动宽带服务
- **全IP网络**：采用分组交换，无需电路交换

!!! note "关键技术"
    - OFDM（正交频分复用）：提高频谱效率
    - MIMO（多输入多输出）：利用多天线提高传输速率

### 5G（NR）

**新空口**（NR，New Radio）是第五代移动通信技术，主要特点：

- **超高速率**：峰值速率可达 $10\text{Gbps}$ 以上
- **超低时延**：端到端时延可低至 $1\text{ms}$
- **海量连接**：每平方公里可支持 $100$ 万个连接
- **高可靠性**：可用于工业控制、自动驾驶等

!!! info "5G应用场景"
    - eMBB（增强移动宽带）：高清视频、VR/AR
    - uRLLC（超可靠低时延通信）：自动驾驶、工业控制
    - mMTC（海量机器通信）：物联网、智慧城市

## 移动互联网

### 移动互联网络架构

```
终端设备 → 无线接入网（基站） → 核心网 → 互联网
```

### 无线接入技术

#### Wi-Fi

- **频段**：$2.4\text{GHz}$（$2400\text{MHz} \sim 2483.5\text{MHz}$）和 $5\text{GHz}$
- **标准**：802.11b/g/n/ac/ax
- **速率**：802.11ax（Wi-Fi 6）可达 $9.6\text{Gbps}$

!!! tip "Wi-Fi与移动通信"
    Wi-Fi适用于室内短距离高速接入，移动通信适用于室外移动场景，两者互补。

#### 蓝牙

- **频段**：$2.4\text{GHz}$
- **特点**：短距离、低功耗、低成本
- **应用**：耳机、音箱、智能手表、IoT设备

#### 物联网（NB-IoT、LoRa）

- **NB-IoT**：基于蜂窝网络，支持海量低功耗设备接入
- **LoRa**：远距离、低功耗，适合广域物联网

## 电磁辐射与安全

### 电磁辐射的类型

- **电离辐射**：X射线、γ射线等，能量高，可电离原子，对人体有较大伤害
- **非电离辐射**：无线电波、红外线、可见光等，能量低，不会电离原子

!!! note "移动通信辐射"
    移动通信使用的无线电波属于非电离辐射，能量较低，不会破坏人体组织。

### 电磁辐射标准

各国对电磁辐射有严格的限制标准。我国规定：

| 场景 | 功率密度限值 |
|:----:|:------------:|
| 公众暴露 | $<0.4\text{W/m}^2$（24GHz以上）|
| 职业暴露 | $<2\text{W/m}^2$（24GHz以上）|

基站和手机在上市前必须通过电磁辐射安全认证。

### 安全使用建议

- 使用耳机或免提，减少手机贴近头部的时间
- 选择符合国家标准的产品
- 信号弱时尽量减少通话时间

## 知识要点总结

!!! note "核心知识"
    1. **移动通信**：蜂窝网络结构，越区切换
    2. **频段应用**：低频覆盖广，高频容量大
    3. **4G/5G**：高速率、低时延、海量连接
    4. **移动互联网**：Wi-Fi、蓝牙、物联网等多种接入方式
    5. **电磁安全**：无线电波属于非电离辐射，有严格安全标准

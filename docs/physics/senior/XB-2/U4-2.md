## 概说

电磁波是高中物理选择性必修二第四章的重要组成部分。本节主要研究电磁波的产生与传播、电磁波的特征等基本知识。

## 麦克斯韦电磁场理论

### 电磁场理论的两大支柱

**麦克斯韦电磁场理论**是经典电动力学的基础，主要包含两个核心观点：

1. **变化的磁场产生电场**：变化的磁场在其周围空间激发电场，这种电场与静止电荷激发的静电场不同，是涡旋电场，其电场线是闭合的。

2. **变化的电场产生磁场**：变化的电场在其周围空间激发磁场，这种磁场与恒定电流激发的磁场相同，是涡旋磁场。

!!! note "关键理解"
    静电场和恒定磁场的理论是麦克斯韦电磁场理论在特殊情况下的体现，而变化的电磁场则揭示了电场和磁场的内在联系。

### 恒定场与变化场

| 情形 | 磁场与电场的关系 |
|:----:|:---------------:|
| **恒定不变** | 恒定的磁场不产生电场，恒定的电场不产生磁场 |
| **均匀变化** | 均匀变化的磁场产生恒定的电场，均匀变化的电场产生恒定的磁场 |
| **周期性变化** | 周期性变化的磁场产生同频率周期性变化的电场，反之亦然 |

### 电磁场的形成

当空间中某处的电场发生周期性变化时，会在周围空间产生周期性变化的磁场；这个变化的磁场又会在更远的空间产生周期性变化的电场。如此交替激发，形成一个不可分割的**电磁场**。这种变化的电磁场在空间传播，就形成了**电磁波**。

## 电磁波

### 什么是电磁波

**电磁波**是变化的电磁场在空间中的传播。电磁波的传播不需要介质，可以在真空中传播。

!!! tip "重要结论"
    光是一种电磁波，无线电波、X射线、γ射线等都是电磁波，它们本质相同，只是频率不同。

### 电磁波的产生

变化的电磁场从产生区域向外传播。理论上，任何加速运动的电荷都可以产生电磁波，但在实际应用中，通常采用**振荡电路**来产生特定频率的电磁波。

#### 开放电路与封闭电路

- **封闭电路**：电磁场几乎全部局限在电路内部，能量难以辐射出去
- **开放电路**：把电容器的两个极板分开、把线圈的匝数减少，使电场和磁场散布到更大的空间，能够更有效地向外辐射电磁波

!!! info "实际应用"
    无线电广播、电视、手机通信等都是利用开放电路发射电磁波的。

### 电磁波的特征量

电磁波的三个特征量之间存在如下关系：

$$v = \lambda f$$

其中：
- $v$：电磁波的传播速度
- $\lambda$：电磁波的波长
- $f$：电磁波的频率

!!! note "频率的独立性"
    电磁波在空间传播时，其频率由波源决定，始终保持不变。当电磁波从一种介质进入另一种介质时，频率不变，波长和波速改变。

### 赫兹实验

#### 实验背景

1887年至1888年间，德国物理学家**海因里希·赫兹**（Heinrich Hertz）首次用实验证实了电磁波的存在。

#### 实验装置

赫兹实验装置包括：

- **感应线圈**：产生高频高压脉冲
- **发射天线**：由两根金属棒组成，中间留有间隙（火花隙），形成开放电路
- **接收天线**：同样为开放电路，金属棒也有间隙

```
        ┌─┐       ┌─┐
   ─────┤ ├───┐┌──┤ ├─────
        └─┘   │└  └─┘
              ↑
            火花隙
```

#### 实验原理

1. 感应线圈产生的高频高压脉冲使发射天线的金属棒间产生火花放电
2. 火花放电使金属棒中的电荷做周期性加速运动
3. 根据麦克斯韦理论，周期性加速的电荷会向外辐射电磁波

#### 实验现象

- 接收天线的金属球间隙处产生火花
- 当发射天线产生火花时，接收天线一定距离内也会产生火花
- 发射天线与接收天线距离改变时，火花强度会变化

#### 实验结论

1. **电磁波确实存在**：电磁波可以从发射天线传播到接收天线
2. **电磁波具有反射、折射、干涉、衍射等性质**：与光波相同
3. **验证了麦克斯韦电磁场理论**：变化的磁场产生电场，变化的电场产生磁场

!!! note "历史意义"
    赫兹实验不仅证实了电磁波的存在，还为后来的无线电通信奠定了基础。电磁波的发现是人类历史上的重要里程碑。

### 电磁波的速度

在真空中，电磁波的速度等于光速：

$$c = \frac{1}{\sqrt{\varepsilon_0\mu_0}} = 3\times10^8\text{m/s}$$

其中 $\varepsilon_0$ 为真空介电常数，$\mu_0$ 为真空磁导率。这与光速相同，说明光确实是一种电磁波。

!!! tip "记忆口诀"
    "光速不变三亿秒，波速随波源；介质变化波速变，频率永远不变。"

## 知识要点总结

!!! note "核心知识"
    1. **麦克斯韦电磁场理论**：变化的磁场产生电场，变化的电场产生磁场
    2. **电磁波**：变化的电磁场在空间中的传播，不需要介质
    3. **赫兹实验**：首次实验证实电磁波的存在，验证麦克斯韦理论
    4. **光速公式**：$c = \lambda f$，真空中 $c = 3\times10^8\text{m/s}$
    5. **开放电路**：更有效地向外辐射电磁波

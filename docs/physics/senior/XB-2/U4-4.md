## 概说

无线电波的发射、传播与接收是高中物理选择性必修二第四章的核心内容。本节主要研究无线电波的发射原理、传播方式以及接收原理。

## 无线电波的发射

### 发射的基本原理

无线电波的发射是通过**振荡电路**产生高频电磁振荡，然后通过**天线**向空间辐射出去。

### 基本发射电路

无线电波发射的基本电路包括以下几个部分：

#### 振荡电路

**振荡电路**产生高频电磁振荡信号。常见的振荡电路有：

- **LC振荡器**：利用电感和电容的周期性储能-放能过程产生振荡
- **晶体振荡器**：利用石英晶体的压电效应产生稳定频率的振荡

#### 调制电路

**调制电路**将低频音频信号加载到高频载波上。调制方式包括调幅（AM）和调频（FM）。

#### 高频放大器

**高频放大器**将调制后的信号进行功率放大，使其有足够的能量通过天线辐射出去。

#### 天线

**天线**将高频电流转换为电磁波向空间辐射。天线的长度与电磁波的波长有关，通常为波长的 $1/4$ 或 $1/2$。

### 调制

**调制**是把信息加载到电磁波上的过程。需要传输的信号（如声音、图像）的频率较低，无法直接辐射，需要加载到高频载波上。

#### 调幅（AM）

**调幅**是使高频电磁波的振幅随信号变化。音频信号放大后，加在高频载波的振幅上，使载波的振幅随音频信号变化。

$$\text{调幅波} = A_m[1 + k \cdot m(t)]\cos(\omega_c t)$$

其中 $m(t)$ 为调制信号，$k$ 为调制系数。

!!! info "应用"
    中波广播、短波广播一般使用调幅方式。

#### 调频（FM）

**调频**是使高频电磁波的频率随信号变化。音频信号放大后，加在高频载波的频率上，使载波的频率随音频信号变化。

$$\omega = \omega_c[1 + k \cdot m(t)]$$

!!! info "应用"
    调频广播、电视伴音一般使用调频方式。调频信号抗干扰能力强，音质好。

### 发射系统框图

```
音频信号 → 调制器 → 高频载波（已调波） → 放大器 → 天线 → 空间传播
```

## 无线电波的传播

### 传播方式

无线电波主要有三种传播方式：

| 传播方式 | 特点 | 应用 |
|:--------:|:----:|:----:|
| **地波传播** | 沿地面传播，绕过障碍物能力强 | 长波、中波广播 |
| **天波传播** | 被电离层反射，可传播很远 | 短波广播 |
| **空间波传播** | 直线传播，传播距离有限 | 电视、调频广播、移动通信 |

### 地波传播

- 波长较长（约 $1000\text{m}$ 以上）的无线电波能够绕过地面障碍物，沿地球表面传播
- 这种传播方式稳定，但能量会被地面吸收，传播距离有限
- 主要用于长波、中波广播

### 天波传播

- 波长较短（约 $10\text{m} \sim 100\text{m}$）的无线电波能被高空电离层反射
- 电离层位于距地面 $60\text{km} \sim 400\text{km}$ 的高空，由带电粒子组成
- 利用天波传播，无线电波可以传播到很远的地方，甚至环绕地球
- 主要用于短波广播

!!! note "电离层特性"
    电离层的高度和密度随昼夜、季节、太阳活动等因素变化，影响无线电波的传播。

### 空间波传播

- 波长更短（如米波、分米波）的无线电波沿直线传播
- 传播距离受地球曲率限制，适用于短距离通信
- 主要用于电视、调频广播、移动通信

!!! tip "传播方式选择"
    不同波长的无线电波采用不同的传播方式：
    - 长波、中波：地波为主
    - 短波：天波为主
    - 米波、分米波：空间波为主

## 无线电波的接收

### 接收的基本原理

无线电波的接收是利用**电谐振**现象。当接收电路的振荡频率与要接收的电磁波频率相同时，接收电路中产生的振荡电流最强，这种现象称为**电谐振**。

### 调谐

**调谐**是从众多电磁波中选取需要接收的特定频率的电磁波的过程。

通过调节接收电路的可变电容或电感，使接收电路的固有频率等于要接收的电磁波频率，发生电谐振，选择出该频率的信号。

### 解调

**解调**是从高频电磁波中取出音频、视频信号的过程，是调制的逆过程。

| 调制方式 | 解调方式 |
|:--------:|:--------:|
| 调幅（AM） | 检波 |
| 调频（FM） | 鉴频 |

#### 检波

检波是从调幅波中取出调制信号的过程。通过二极管的单向导电性，去除载波，只保留包络信号，再经滤波、放大得到音频信号。

#### 鉴频

鉴频是从调频波中取出调制信号的过程。通过频率-电压转换，将频率变化转化为电压变化，恢复原始信号。

### 接收系统框图

```
天线 → 调谐电路 → 高频放大 → 检波/鉴频 → 音频放大 → 扬声器/显示器
```

## 知识要点总结

!!! note "核心知识"
    1. **调制**：把低频信号加载到高频载波上
       - 调幅（AM）：振幅随信号变化
       - 调频（FM）：频率随信号变化
    2. **传播方式**：
       - 地波：长波、中波，沿地面传播
       - 天波：短波，被电离层反射
       - 空间波：米波、分米波，直线传播
    3. **接收原理**：
       - 电谐振：固有频率等于电磁波频率时电流最强
       - 调谐：选择特定频率的电磁波
       - 解调：从高频波中取出原始信号（检波、鉴频）

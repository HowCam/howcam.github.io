## 概说

电磁波谱是高中物理选择性必修二第四章的重要内容。本节主要研究电磁波按波长或频率排列形成的谱系，以及各波段电磁波的特点和应用。

## 电磁波谱

### 什么是电磁波谱

**电磁波谱**是按电磁波的波长或频率的大小，把所有电磁波依次排列成的谱。按照波长从长到短（或频率从低到高），电磁波谱的顺序为：

**无线电波 → 红外线 → 可见光 → 紫外线 → X射线 → γ射线**

!!! note "连续谱"
    电磁波谱是一个连续谱，各波段之间没有严格的界限。

### 各波段一览表

| 波段 | 波长范围 | 频率范围 | 主要来源 |
|:----:|:--------:|:--------:|:--------:|
| **无线电波** | $>1\text{mm}$ | $<3\times10^{11}\text{Hz}$ | 振荡电路、天线 |
| **红外线** | $1\text{mm} \sim 770\text{nm}$ | $3\times10^{11} \sim 3.9\times10^{14}\text{Hz}$ | 热辐射、物体 |
| **可见光** | $770\text{nm} \sim 380\text{nm}$ | $3.9\times10^{14} \sim 8\times10^{14}\text{Hz}$ | 太阳、发光体 |
| **紫外线** | $380\text{nm} \sim 10\text{nm}$ | $8\times10^{14} \sim 3\times10^{16}\text{Hz}$ | 太阳、汞灯 |
| **X射线** | $10\text{nm} \sim 0.01\text{nm}$ | $3\times10^{16} \sim 3\times10^{19}\text{Hz}$ | X射线管 |
| **γ射线** | $<0.01\text{nm}$ | $>3\times10^{19}\text{Hz}$ | 放射性物质 |

## 无线电波

### 无线电波概述

无线电波是波长最长的电磁波，按波长可分为长波、中波、短波、微波等。

### 无线电波的应用

无线电波主要用于通信、广播、雷达、导航等。

!!! info "常见应用频段"
    - 调幅广播（AM）：频率 $535\text{kHz} \sim 1605\text{kHz}$，波长约 $187\text{m} \sim 560\text{m}$
    - 调频广播（FM）：频率 $88\text{MHz} \sim 108\text{MHz}$，波长约 $2.8\text{m} \sim 3.4\text{m}$
    - 电视：频率 $48.5\text{MHz} \sim 958\text{MHz}$
    - 手机通信：频率约 $800\text{MHz} \sim 2600\text{MHz}$

### 无线电波的波长与频率

根据 $c = \lambda f$：

$$\lambda = \frac{c}{f}$$

- 频率越高，波长越短
- 频率越低，波长越长

## 红外线

### 红外线的特点

红外线的波长比可见光的红光还长，不能引起视觉。红外线具有显著的热效应，物体吸收红外线后温度升高。

!!! note "主要特点"
    1. 热效应显著：可用于加热、烘干
    2. 穿透能力强：能穿透云雾和烟尘
    3. 可用于遥感：红外测温、红外成像

### 红外线的应用

- 加热设备：烤箱、取暖器、热疗仪
- 探测设备：红外测温、红外遥感、夜视仪
- 通信领域：红外遥控、数据传输

## 可见光

### 可见光概述

可见光是波长约 $380\text{nm} \sim 770\text{nm}$ 的电磁波，它能引起人眼的视觉。按波长从长到短，可分为红、橙、黄、绿、蓝、靛、紫七种颜色。

| 颜色 | 波长范围 | 频率范围 |
|:----:|:--------:|:--------:|
| 红 | $770\text{nm} \sim 620\text{nm}$ | $3.9\times10^{14} \sim 4.8\times10^{14}\text{Hz}$ |
| 橙 | $620\text{nm} \sim 590\text{nm}$ | $4.8\times10^{14} \sim 5.1\times10^{14}\text{Hz}$ |
| 黄 | $590\text{nm} \sim 570\text{nm}$ | $5.1\times10^{14} \sim 5.3\times10^{14}\text{Hz}$ |
| 绿 | $570\text{nm} \sim 500\text{nm}$ | $5.3\times10^{14} \sim 6.0\times10^{14}\text{Hz}$ |
| 蓝 | $500\text{nm} \sim 450\text{nm}$ | $6.0\times10^{14} \sim 6.7\times10^{14}\text{Hz}$ |
| 靛 | $450\text{nm} \sim 430\text{nm}$ | $6.7\times10^{14} \sim 7.0\times10^{14}\text{Hz}$ |
| 紫 | $430\text{nm} \sim 380\text{nm}$ | $7.0\times10^{14} \sim 8.0\times10^{14}\text{Hz}$ |

!!! tip "记忆口诀"
    红橙黄绿蓝靛紫，波长从长到短；红光最长，紫光最短。

## 紫外线

### 紫外线的特点

紫外线的波长比可见光的紫光还短，不能引起视觉。紫外线具有显著的化学作用和荧光效应。

!!! note "主要特点"
    1. 化学作用强：能使照像底片感光
    2. 荧光效应：能激发荧光物质发光
    3. 杀菌消毒：能破坏微生物的DNA

### 紫外线的应用

- 杀菌消毒：医疗消毒、水处理
- 防伪标识：验钞机、防伪标签
- 促进合成：维生素D的合成（适度照射）

!!! warning "防护"
    过量紫外线照射会伤害皮肤和眼睛，应注意防护。

## X射线和γ射线

### X射线

X射线波长约 $10\text{nm} \sim 0.01\text{nm}$，穿透能力较强。主要应用于：

- 医学诊断：X光透视、CT扫描
- 工业探伤：检测材料内部缺陷

### γ射线

γ射线波长最短，穿透能力最强。主要应用于：

- 医学治疗：放射治疗癌症
- 工业检测：工业探伤
- 食品保鲜：杀灭细菌

!!! warning "辐射防护"
    X射线和γ射线对人体有害，长期接触可能导致癌症。医疗和工业应用中需要采取严格的防护措施。

## 知识要点总结

!!! note "核心知识"
    1. **电磁波谱**：无线电波 → 红外线 → 可见光 → 紫外线 → X射线 → γ射线
    2. **波长与频率的关系**：$\lambda = c/f$，波长从长到短，频率从低到高
    3. **各波段特点**：
       - 无线电波：用于通信、广播
       - 红外线：热效应显著，用于加热、遥感
       - 可见光：人眼可见，七色光
       - 紫外线：化学作用强，用于消毒
       - X射线和γ射线：穿透力强，用于医疗

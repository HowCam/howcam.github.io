# 细胞的增殖

## 细胞增殖的意义

细胞增殖是重要的细胞生命活动，是生物体生长、发育、繁殖、遗传的基础。

细胞增殖的方式：有丝分裂、无丝分裂、减数分裂。

## 有丝分裂

### 细胞周期

**连续分裂的细胞**，从第一次分裂完成开始，到下一次分裂完成时为止，为一个细胞周期。

!!! question "关于"连续分裂"概念的相关考点"

    高度分化的细胞无法连续分裂，例如：洋葱表皮细胞、哺乳动物成熟红细胞、蛙的红细胞、精细胞、浆细胞等。

    特别地，十二指肠上皮细胞、肝细胞可以连续分裂。

一个细胞周期分为两个阶段：**分裂间期**和**分裂期**。

【分裂间期】

- G1期：蛋白质合成，为S期做准备
- S期：DNA复制
- G2期：蛋白质合成，为分裂期做准备
- 细胞适度生长

!!! question "怎么考？"

    ｜—a—｜———b———｜—c—｜

    【b+c是一个周期（间期在前，间期时间长）】

### 分裂间期

- 完成了DNA复制和有关蛋白质的合成
- 细胞有适度生长
- 染色体复制后的DNA分子数目加倍，但染色体数目不变

### 分裂期

**有丝分裂口诀：**

- 膜仁消失现两体（前期）
- 形定数晰赤道齐（中期）
- 粒裂数增均两极（后期）
- 两消两现新壁建（末期）

**前期**

- 染色质丝螺旋缠绕，缩短变粗，成为染色体
- 每个染色体包括两条并列的姐妹染色单体
- 核仁逐渐解体，核膜逐渐消失
- 细胞两极发出纺锤丝，形成纺锤体

**中期**

- 所有染色体的着丝粒排列在赤道板上
- 细胞形态固定，数目清晰
- 是观察和计数染色体的最佳时期

**后期**

- 每个着丝粒分裂，姐妹染色单体分开，成为两条子染色体
- 纺锤丝牵引子染色体分别向细胞两极移动
- 细胞两极的染色体数目加倍

**末期**

- 染色体变为染色质丝
- 核膜、核仁重新出现
- 纺锤体消失
- 细胞板扩展为细胞壁（植物细胞）

!!! note "动植物细胞有丝分裂的区别"

    | 特征 | 植物细胞 | 动物细胞 |
    |------|----------|----------|
    | 前期 | 两极发出**纺锤丝**形成纺锤体 | 中心体发出**星射线**形成纺锤体 |
    | 末期 | **细胞板**扩展为细胞壁 | 细胞膜从中部**向内凹陷**，缢裂成两个子细胞 |

### 数量变化

有丝分裂中染色体、DNA、染色单体的数量变化：

| 时期 | 染色体数 | DNA分子数 | 染色单体数 |
|------|----------|-----------|------------|
| 间期（复制前） | 2n | 2c | 0 |
| 间期（复制后） | 2n | 4c | 4n |
| 前期 | 2n | 4c | 4n |
| 中期 | 2n | 4c | 4n |
| 后期 | 4n | 4c | 0 |
| 末期（结束） | 2n | 2c | 0 |

### 有丝分裂的意义

- 将亲代细胞的染色体经过复制后，精确地平均分配到两个子细胞中
- 保持了亲子代之间遗传性状的稳定性

## 无丝分裂

**特点**：无染色体和纺锤体的出现，DNA复制后细胞直接分裂。

**过程**：细胞核延长 → 核中央凹陷 → 细胞缢裂成两个子细胞

**代表细胞**：蛙的红细胞

**意义**：无丝分裂不产生染色体，保持了遗传物质的连续性

## 减数分裂

### 减数分裂的概念

进行有性生殖的生物，在产生成熟生殖细胞时进行的染色体数目减半的细胞分裂。

!!! note "关键点"

    - 范围：只有进行有性生殖的生物才有减数分裂
    - 位置：生殖器官（睾丸、卵巢）
    - 特点：染色体只复制一次，细胞分裂两次
    - 结果：成熟生殖细胞中染色体数目比原始生殖细胞减少一半

### 减数分裂的过程

**减数第一次分裂**

| 时期 | 主要特征 |
|------|----------|
| 间期 | 染色体复制（DNA复制和有关蛋白质合成），同源染色体联会形成四分体 |
| 前期Ⅰ | 同源染色体联会，形成四分体；非姐妹染色单体之间可能发生交叉互换 |
| 中期Ⅰ | 四分体排列在赤道板两侧 |
| 后期Ⅰ | 同源染色体分离，非同源染色体自由组合 |
| 末期Ⅰ | 细胞质分裂，形成两个子细胞 |

**减数第二次分裂**

| 时期 | 主要特征 |
|------|----------|
| 前期Ⅱ | 染色体散乱分布 |
| 中期Ⅱ | 染色体的着丝粒排列在赤道板上 |
| 后期Ⅱ | 着丝粒分裂，姐妹染色单体分开 |
| 末期Ⅱ | 细胞质分裂，形成四个子细胞 |

### 染色体的行为

**同源染色体与非同源染色体**

- **同源染色体**：形态、大小一般相同，一条来自父方，一条来自母方，在减数分裂中联会的两条染色体
- **非同源染色体**：形态、大小不同，在减数分裂中不联会的染色体

**四分体**

四分体 = 一对同源染色体 = 4条染色单体

交叉互换：发生在非姐妹染色单体之间

**减数分裂中染色体、DNA、染色单体的数量变化**

以精形成为例：

| 时期 | 染色体数 | DNA分子数 | 染色单体数 |
|------|----------|-----------|------------|
| 精原细胞 | 2n | 2c | 0 |
| 初级精母细胞 | 2n | 4c | 4n |
| 次级精母细胞（前期Ⅱ/中期Ⅱ） | n | 2c | 2n |
| 精细胞 | n | c | 0 |

### 精子和卵细胞的形成

**精子形成**

精原细胞 → 初级精母细胞 → 次级精母细胞 → 精细胞 → 精子

- 一个精原细胞经过减数分裂产生 **四个** 精细胞
- 精细胞经过变形形成精子

**卵细胞形成**

卵原细胞 → 初级卵母细胞 → 次级卵母细胞 + 第一极体 → 卵细胞 + 第二极体

- 一个卵原细胞经过减数分裂产生 **一个** 卵细胞和三个极体
- 极体最终退化消失

!!! warning "注意"

    一个初级卵母细胞只能产生一个卵细胞，而一个初级精母细胞能产生四个精细胞。

### 受精作用

**概念**：精子与卵细胞融合形成受精卵的过程。

**过程**：顶体反应 → 透明带反应 → 卵细胞膜反应 → 雌雄原核融合

**意义**：

1. 受精卵中染色体数目恢复为体细胞中的染色体数目
2. 受精卵的遗传物质一半来自精子，一半来自卵细胞
3. 受精作用实现了基因的重新组合，增加了后代的变异范围

## 有丝分裂与减数分裂的比较

| 比较项目 | 有丝分裂 | 减数分裂 |
|----------|----------|----------|
| 分裂细胞 | 体细胞 | 原始生殖细胞 |
| 复制次数 | 1次 | 1次 |
| 分裂次数 | 1次 | 2次 |
| 同源染色体行为 | 无联会、无交叉互换 | 联会、交叉互换 |
| 子细胞染色体数 | 与亲代相同 | 是亲代的一半 |
| 子细胞类型 | 体细胞 | 生殖细胞 |
| 相同点 | 染色体复制一次，分裂一次；染色体出现、纺锤体出现和消失，核膜、核仁消失和重建 | 染色体复制一次，分裂一次；染色体出现、纺锤体出现和消失，核膜、核仁消失和重建 |

## 细胞不能无限长大的原因

### 细胞体积的局限性

细胞体积的相对表面积（表面积与体积的比值）随细胞增大而减小。

!!! note "相关计算"

    立方体模型：
    
    - 边长为 a：表面积 = $6a^2$，体积 = $a^3$
    - 相对表面积 = $\frac{表面积}{体积} = \frac{6a^2}{a^3} = \frac{6}{a}$
    
    结论：细胞体积增大时，相对表面积减小，物质运输效率降低。

### 细胞不能无限长大的原因

1. **细胞表面积与体积的限制**
   - 细胞体积增大，细胞表面积和体积的比值（相对表面积）减小
   - 细胞与外界进行物质交换的效率降低

2. **细胞核的控制能力有限**
   - 细胞核中的DNA不会随细胞体积增大而无限增加
   - 细胞核的控制能力存在一定范围，无法满足过大细胞的调控需求

3. **细胞内物质运输的效率**
   - 细胞内部需要通过扩散运输物质
   - 细胞过大时，物质运输距离增加，效率降低

!!! note "结论"

    细胞不能无限长大，需要通过分裂来增加细胞数量，从而保持合适的表面积与体积比例，保证物质交换和信息传递的效率。

## 相关模拟实验

### 细胞大小与物质运输的关系

**实验目的**：探究细胞大小与物质运输效率之间的关系。

**实验原理**：利用琼脂块（模拟细胞）吸收酚酞溶液（模拟营养物质），测量不同大小的琼脂块运输物质的效率。

**实验步骤**：

1. 配制含有酚酞的琼脂溶液，凝固成不同边长的立方体琼脂块
2. 将琼脂块放入NaOH溶液中浸泡
3. 相同时间后取出，切开琼脂块
4. 测量NaOH扩散的深度

**实验结果**：

| 琼脂块边长 | 扩散深度 | 着色部分占比 | 结论 |
|------------|----------|--------------|------|
| 小 | 扩散比例大 | 几乎完全着色 | 物质运输效率高 |
| 大 | 扩散比例小 | 仅表层着色 | 物质运输效率低 |

!!! note "实验结论"

    琼脂块体积越大，相对表面积越小，NaOH扩散进入的比例越小，物质运输效率越低。

    **推论**：细胞体积不能无限增大的原因之一是细胞体积增大时，细胞的相对表面积减小，物质运输效率降低。

### 有丝分裂的观察实验

**实验目的**：观察植物细胞有丝分裂的过程。

**实验材料**：洋葱根尖（分生区细胞分裂旺盛）

**实验步骤**：

1. **解离**：将根尖放入解离液（盐酸和酒精混合液）中，使细胞分散开来
2. **漂洗**：用清水漂洗根尖，去除解离液
3. **染色**：用碱性染料（如龙胆紫或醋酸洋红）染色，使染色体着色
4. **制片**：将根尖放在载玻片上，盖上盖玻片，轻压使细胞分散
5. **观察**：先用低倍镜找到分生区，再换高倍镜观察

**观察结果**：可以看到细胞分裂各时期的染色体形态变化

**有丝分裂各时期特征**：

- 前期：染色体出现，纺锤体形成
- 中期：染色体排列在赤道板上
- 后期：着丝粒分裂，染色体移向两极
- 末期：染色体恢复成染色质，出现细胞板

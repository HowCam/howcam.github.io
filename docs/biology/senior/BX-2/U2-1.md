## 减数分裂的概念

减数分裂是进行有性生殖的生物，在产生成熟生殖细胞时进行的染色体数目减半的细胞分裂。

!!! note "关键特征"

    - **范围**：只有进行有性生殖的生物才有减数分裂

    - **位置**：生殖器官（睾丸、卵巢）

    - **特点**：染色体只复制一次，细胞分裂两次

    - **结果**：成熟生殖细胞中染色体数目比原始生殖细胞减少一半

## 减数分裂的过程

减数分裂分为两个阶段：**减数第一次分裂**（减Ⅰ）和**减数第二次分裂**（减Ⅱ）。

### 减数第一次分裂

| 时期 | 主要特征 |
|------|----------|
| 减数分裂前的间期 | 染色体复制（DNA复制和有关蛋白质合成）（中心体复制） |
| 前期Ⅰ | 同源染色体联会，形成四分体；非姐妹染色单体之间可能发生交叉互换 |
| 中期Ⅰ | 同源染色体（四分体）整齐排列在赤道板**两侧** |
| 后期Ⅰ | 同源染色体分离，非同源染色体自由组合 |
| 末期Ⅰ | 细胞质分裂，形成两个子细胞 |

### 减数第二次分裂

| 时期 | 主要特征 |
|------|----------|
| 前期Ⅱ | 染色体散乱分布 |
| 中期Ⅱ | 染色体的着丝粒排列在赤道板上 |
| 后期Ⅱ | 着丝粒分裂，姐妹染色单体分开 |
| 末期Ⅱ | 细胞质分裂，形成四个子细胞 |

!!! tip "记忆口诀"

    - 减Ⅰ：同源联会、交叉互换；排列赤道、自由组合；同源分离

    - 减Ⅱ：类似有丝，中排赤道，后分两极

## 染色体、同源染色体与四分体

### 同源染色体与非同源染色体

- **同源染色体**：形态、大小一般相同，一条来自父方，一条来自母方，在减数分裂中**联会**的两条染色体

- **非同源染色体**：形态、大小不同，在减数分裂中不联会的染色体

!!! note "判断方法"

    同源染色体的判断依据：

    1. 一条来自父方，一条来自母方

    2. 形态、大小一般相同（性染色体除外）

    3. 在减数分裂中能联会

### 四分体

- **四分体** = 一对联会的同源染色体 = 4条染色单体

- **交叉互换**：发生在四分体的非姐妹染色单体之间，导致基因重组

!!! question "相关计算"

    若体细胞中染色体数为 $2n$：

    - 四分体数 = $n$（同源染色体对数）

    - 染色单体数 = $4n$（四分体中）

    - DNA分子数 = $4n$（四分体中）

## 精子和卵细胞的形成

### 精子形成

精原细胞 → 初级精母细胞 → 次级精母细胞 → 精细胞 → 精子

- 一个精原细胞经过减数分裂产生 **四个** 精细胞

- 精细胞经过变形形成精子

- 变形过程中丢去大部分细胞质

### 卵细胞形成

卵原细胞 → 初级卵母细胞 → 次级卵母细胞 + 第一极体 → 卵细胞 + 第二极体

- 一个卵原细胞经过减数分裂产生 **一个** 卵细胞和三个极体

- 极体最终退化消失

!!! warning "精子和卵细胞形成的区别"

    | 比较项目 | 精子形成 | 卵细胞形成 |
    |----------|----------|----------|
    | 产生部位 | 睾丸 | 卵巢 |
    | 原始细胞 | 精原细胞 | 卵原细胞 |
    | 过程 | 精原细胞 → 初级精母细胞 → 次级精母细胞 → 精细胞 → 精子 | 卵原细胞 → 初级卵母细胞 → 次级卵母细胞 + 第一极体 → 卵细胞 + 第二极体 |
    | 子细胞数量 | 1个精原细胞 → 4个精细胞 → 4个精子 | 1个卵原细胞 → 1个卵细胞 + 3个极体 |
    | 细胞质分裂 | 两次分裂均**均等**分裂 | 减Ⅰ、减Ⅱ分裂均**不均等**分裂（形成大的卵细胞和小的极体） |
    | 是否变形 | 精细胞经**变形**形成精子（丢去大部分细胞质） | 卵细胞**不需要**变形 |
    | 营养物质 | 细胞质中营养物质少 | 细胞质中营养物质丰富（为早期发育提供营养） |
    | 极体 | 无极体（不产生极体） | 产生**3个**极体（最终退化消失） |

## 减数分裂中染色体、DNA的数量变化

以精形成为例：

| 时期 | 染色体数 | DNA分子数 | 染色单体数 |
|------|----------|-----------|------------|
| 精原细胞 | 2n | 2c | 0 |
| 初级精母细胞 | 2n | 4c | 4n |
| 次级精母细胞（前期Ⅱ/中期Ⅱ） | n | 2c | 2n |
| 次级精母细胞（后期Ⅱ） | 2n | 2c | 0 |
| 精细胞 | n | c | 0 |

!!! note "数量变化规律"

    - 染色体数目：$2n \rightarrow n \rightarrow 2n \rightarrow n$（减半发生在减Ⅰ末期）

    - DNA数目：$2c \rightarrow 4c \rightarrow 2c \rightarrow c$（减半发生在减Ⅱ末期）

    - 染色单体数目：$0 \rightarrow 4n \rightarrow 2n \rightarrow 0$（减半发生在减Ⅱ后期）

## 受精作用

### 概念

受精作用是精子与卵细胞融合形成受精卵的过程。

### 受精过程

1. **顶体反应**：精子获能后，与卵相遇时释放顶体酶，溶解卵细胞外的放射冠
2. **透明带反应**：精子穿过透明带，与卵细胞膜接触。这是防止多精子入卵的第一道屏障
3. **卵细胞膜反应**：精子入卵后，卵细胞膜立即发生复杂的生理变化，拒绝其他精子进入。这是防止多精子入卵的第二道屏障
4. **雌雄原核形成**：精子入卵后形成雄原核，卵细胞完成减数分裂Ⅱ形成雌原核
5. **雌雄原核融合**：两个原核相互靠近，融合形成受精卵

!!! info "受精标志"

    受精的标志：在透明带和卵细胞膜之间观察到两个极体

    受精完成的标志：雌雄原核融合

### 受精作用的实质

雄原核与雌原核融合。

### 受精作用的意义

1. **恢复染色体数目**：受精卵中染色体数目恢复为体细胞中的染色体数目
2. **遗传物质的来源**：受精卵的遗传物质一半来自精子，一半来自卵细胞
3. **增加变异范围**：受精作用实现了基因的重新组合，增加了后代的变异范围
4. **维持染色体数目恒定**：对于维持每种生物前后代体细胞中染色体数目的恒定具有重要意义

## 有丝分裂与减数分裂的比较

!!! note "比较"

    | 比较项目 | 有丝分裂 | 减数分裂 |
    |----------|----------|----------|
    | 分裂细胞类型 | 体细胞 | 原始生殖细胞 |
    | 染色体复制次数 | 1次 | 1次 |
    | 细胞分裂次数 | 1次 | 2次 |
    | 同源染色体行为 | 无联会、无交叉互换 | 联会、交叉互换 |
    | 子细胞染色体数 | 与亲代相同 | 是亲代的一半 |
    | 子细胞类型 | 体细胞 | 生殖细胞 |
    | 相同点 | 染色体复制一次，分裂一次；染色体出现、纺锤体出现和消失，核膜、核仁消失和重建 |

### 辨析方法

!!! question "如何判断细胞分裂方式？"

    1. **看有无同源染色体联会、四分体、交叉互换**

       - 有 → 减数第一次分裂

       - 无 → 可能是有丝分裂或减数第二次分裂

    2. **看有无同源染色体**

       - 有同源染色体且不联会 → 有丝分裂

       - 无同源染色体 → 减数第二次分裂

    3. **看染色体数目变化**

       - 染色体数目不变 → 有丝分裂

       - 染色体数目减半 → 减数第一次分裂或结束时

## 配子中染色体组合的多样性

### 非同源染色体自由组合

在减数第一次分裂后期，同源染色体分离，非同源染色体自由组合。

若生物体有 $n$ 对同源染色体，则理论上可产生 $2^n$ 种配子。

!!! example "例题"

    人的体细胞中有 23 对染色体，则理论上可以产生 $2^{23} = 8,388,608$ 种配子。

### 交叉互换

在减数第一次分裂前期，四分体中非姐妹染色单体之间可能发生交叉互换，进一步增加配子的多样性。

!!! note "配子多样性的原因"

    1. **非同源染色体的自由组合**：$2^n$ 种组合

    2. **同源染色体间的交叉互换**：产生新的基因组合

    3. **雄性可产生大量精子**：进一步增加了后代的多样性

## 观察蝗虫精母细胞减数分裂实验

### 实验目的

观察减数分裂各时期的特点，识别不同时期的细胞形态。

### 实验材料

蝗虫精巢（精巢内可同时观察到减数分裂各时期的细胞）。

### 实验原理

蝗虫精巢中的精原细胞既进行有丝分裂，又进行减数分裂，可以观察到多种分裂时期的细胞。

### 实验步骤

1. 取材：取蝗虫精巢
2. 固定：用卡诺氏液固定
3. 制作切片：切片厚度为 $5\sim10\mu m$
4. 染色：使用龙胆紫或醋酸洋红染色
5. 观察：先用低倍镜找到相关时期的细胞，再换高倍镜观察

### 观察结果

可以观察到减数分裂各时期的细胞，识别出：
- 精原细胞（有丝分裂）
- 初级精母细胞（减Ⅰ）
- 次级精母细胞（减Ⅱ）
- 精细胞

## 细胞分裂图像辨析

### 有丝分裂各时期特点

| 时期 | 图像特点 |
|------|----------|
| 前期 | 出现染色体和纺锤体，核膜、核仁消失 |
| 中期 | 染色体的着丝粒排列在赤道板上 |
| 后期 | 着丝粒分裂，姐妹染色单体分开，染色体移向两极 |
| 末期 | 染色体变成染色质丝，核膜、核仁出现，细胞质分裂 |

### 减数分裂各时期特点

| 时期 | 图像特点 |
|------|----------|
| 减Ⅰ前期 | 同源染色体联会，形成四分体 |
| 减Ⅰ中期 | 四分体排列在赤道板上 |
| 减Ⅰ后期 | 同源染色体分离，非同源染色体自由组合 |
| 减Ⅱ前期 | 无同源染色体，染色体散乱分布 |
| 减Ⅱ中期 | 无同源染色体，染色体排列在赤道板上 |
| 减Ⅱ后期 | 着丝粒分裂，姐妹染色单体分开 |

!!! tip "快速判断口诀"

    - 有同源染色体且联会 → 减数第一次分裂前期

    - 有同源染色体排列在赤道板两侧 → 减数第一次分裂中期

    - 有同源染色体分离 → 减数第一次分裂后期

    - 无同源染色体 → 减数第二次分裂

    - 有同源染色体但不联会，排列在赤道板上 → 有丝分裂中期

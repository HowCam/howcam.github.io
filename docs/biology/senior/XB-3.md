# 第一章 发酵工程

---

## 第1节 传统发酵技术的应用

### 一、发酵的概念

**发酵（fermentation）**：利用微生物，在适宜条件下，将原料通过微生物的代谢，转化为人类所需产物的过程。

> 1857年，法国微生物学家**巴斯德**（L. Pasteur，1822—1895）通过实验证明酒精发酵由活的酵母菌引起。
> 1897年，科学家发现酶在酵母菌发酵中的作用，逐渐了解发酵本质。

**传统发酵技术**：直接利用原材料中天然存在的微生物，或利用前一次发酵保存的面团、卤汁等中的微生物进行发酵，制作食品的技术。以**混合菌种**的固体发酵及半固体发酵为主，通常是家庭式或作坊式生产。

**发酵工程**：利用微生物的特定功能，规模化生产人类所需产品的综合性生物工程，涉及菌种的选育和培养、产物的分离和提纯等。

---

### 二、传统发酵食品的微生物原理

#### （一）酵母菌——酒精发酵

| 项目 | 内容 |
|------|------|
| 微生物 | 单细胞真菌，**兼性厌氧** |
| 糖类利用 | 能以多种糖类作为营养物质和能量来源 |
| 存在环境 | 含糖量较高的水果、蔬菜表面 |
| 最适温度 | 约 **28℃** |
| 产物 | 乙醇（酒精）+ CO₂ + 能量 |
| 反应式 | C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 能量 |

**应用**：果酒、馒头、面包

> 约9000年前，我国祖先就会利用微生物将谷物、水果发酵为含酒精饮料。

---

#### （二）醋酸菌——醋酸发酵

| 项目 | 内容 |
|------|------|
| 微生物 | **好氧细菌** |
| 条件1（O₂、糖源充足）| 将糖分解为乙酸 |
| 条件2（缺少糖源）| 将乙醇转化为乙醛，再变为乙酸 |
| 最适温度 | **30～35℃** |
| 产物 | 乙酸（醋酸）|
| 反应式（糖源充足）| C₆H₁₂O₆ + 2O₂ → 2CH₃COOH + 2H₂O + 2CO₂ + 能量 |
| 反应式（缺糖源）| C₂H₅OH + O₂ → CH₃COOH + H₂O + 能量 |

**应用**：果醋

---

#### （三）乳酸菌——乳酸发酵

| 项目 | 内容 |
|------|------|
| 微生物 | **厌氧细菌** |
| 常见种类 | 乳酸链球菌、乳酸杆菌 |
| 存在环境 | 空气、土壤、植物体表、人或动物肠道内 |
| 产物 | 乳酸 |
| 反应式 | C₆H₁₂O₆ → 2C₃H₆O₃ + 能量 |

**应用**：泡菜、酸奶、乳制品

---

#### （四）毛霉等——腐乳发酵

| 项目 | 内容 |
|------|------|
| 主要菌种 | **毛霉** |
| 参与微生物 | 酵母、曲霉、毛霉等多种 |
| 发酵原理 | 毛霉等产生**蛋白酶**，将豆腐中蛋白质分解成小分子**肽和氨基酸** |
| 结果 | 味道鲜美，易于消化吸收，便于保存 |

**其他传统发酵食品**：酱、酱油、醋、泡菜、豆豉

---

### 三、泡菜的制作

**原理**：利用植物体表面天然的**乳酸菌**进行乳酸发酵。乳酸质量分数达 **0.4%～0.8%** 时口味品质最佳。

**方法步骤**：
1. 配制盐水（5%～20% 食盐水，煮沸冷却）
2. 新鲜蔬菜洗净切块，晾干，装至**半坛**时放香辛料，**八成满**
3. 灌入盐水，没过全部菜料
4. **水封坛盖**（创造无氧环境）

**关键问题**：
- 水密封目的 → 创造**无氧环境**供乳酸菌发酵
- 只能装八成满 → 发酵产气，防止溢出

**亚硝酸盐**：泡菜腌制过程中产生，膳食过量摄入会中毒死亡；含量先升后降，需控制腌制时间与温度。

---

### 四、果酒和果醋的制作

| | 果酒 | 果醋 |
|--|------|------|
| 主要微生物 | 酵母菌 | 醋酸菌 |
| 氧气需求 | **无氧** | **有氧** |
| 发酵温度 | 18～30℃ | 30～35℃ |
| 发酵时间 | 10～12 d | 7～8 d |
| 原理 | 酒精发酵 | 乙醇→乙酸 |

**果酒制作关键**：留约 **1/3 空间**（供气体膨胀）；每隔约12h拧松瓶盖一次（不是打开）

**果醋制作**：果酒完成后打开瓶盖盖纱布，在30～35℃下通风发酵7～8d

## 第2节 微生物的培养技术及应用

### 一、培养基的配制

**培养基（culture medium）**：按照微生物对营养物质的不同需求，配制供其生长繁殖的营养基质。用于培养、分离、鉴定、保存微生物或积累代谢物。

**类型**：
- 液体培养基（无凝固剂）
- 固体培养基（加入琼脂，实验室最常用）

**培养基共同成分**：水、**碳源**（提供碳元素）、**氮源**（提供氮元素）、无机盐

**实例——牛肉膏蛋白胨培养基（/1000mL）**：

| 组分 | 含量 | 提供营养 |
|------|------|----------|
| 牛肉膏 | 5 g | 碳源、氮源、磷酸盐、维生素 |
| 蛋白胨 | 10 g | 碳源、氮源、维生素 |
| NaCl | 5 g | 无机盐 |

**特殊营养需求**：
- 乳酸杆菌：需添加**维生素**
- 霉菌：培养基调至**酸性**
- 细菌：培养基调至**中性或弱碱性**
- 厌氧微生物：提供**无氧条件**

---

### 二、无菌技术

**消毒（disinfection）**：使用温和方法杀死物体表面或内部**一部分**微生物（不灭芽孢）

**灭菌（sterilization）**：使用强烈理化方法杀死**所有**微生物，包括芽孢和孢子

| 方法 | 条件 | 适用对象 |
|------|------|----------|
| 煮沸消毒 | 100℃，5～6 min | 日常物品 |
| 巴氏消毒 | 63～65℃/30min 或 72～76℃/15s 等 | 牛奶等不耐高温液体 |
| 酒精消毒 | 70% 酒精擦拭 | 双手、器具 |
| 紫外线消毒 | 照射 30 min | 接种室、超净工作台 |
| 高压蒸汽灭菌 | 100 kPa，121℃，15～30 min | 培养基、器械（**最常用**）|
| 干热灭菌 | 160～170℃，1～2 h | 玻璃器皿、金属用具（需保持干燥）|
| 灼烧灭菌 | 酒精灯火焰 | 接种工具、管口 |

> 操作应在**超净工作台**、靠近**酒精灯火焰**进行。

---

### 三、微生物的纯培养

**纯培养物**：由单一个体繁殖所获得的微生物群体。

**单菌落**：分散的微生物在固体培养基表面或内部繁殖，形成的肉眼可见、有一定形态结构的子细胞群体。**一个单菌落一般来源于一个活菌**。

#### 平板划线法

通过接种环在固体培养基表面连续划线，将聚集的菌种逐步稀释分散到培养基表面，经培养获得**单菌落**（纯培养物）。

#### 稀释涂布平板法

将菌液进行**梯度稀释**（10 → 10² → 10³ → ...），取 0.1 mL 涂布在固体培养基表面，长出的单菌落来自单个活菌。

#### 倒平板操作

1. 拔出棉塞，瓶口通过火焰
2. 培养皿打开一条缝隙，倒入培养基（10～20 mL）
3. 立即盖上皿盖
4. 冷却凝固后**倒置**放置（防冷凝水滴落污染）

---

### 四、微生物的选择培养和计数

#### 选择培养基

**原理**：人为提供有利于**目的菌**生长的条件（营养、温度、pH），同时抑制其他微生物。

| 设计 | 目的菌 | 原理 |
|------|--------|------|
| 以**尿素**为唯一氮源 | 分解尿素细菌 | 只有能合成**脲酶**的才能分解尿素产生NH₃ |
| 以**纤维素**为唯一碳源 | 纤维素分解菌 | 只有能产生**纤维素酶**的才能利用纤维素 |
| 70～80℃培养 | 水生栖热菌 | 其他微生物在此高温不能生存 |

> 1973年从美国黄石国家公园**热泉**中筛选出水生栖热菌（Thermus aquaticus），提取出耐高温 **Taq DNA聚合酶**，广泛用于 PCR。

#### 土壤中分解尿素细菌的分离与计数

**步骤**：取土（3～8 cm深处）→ 梯度稀释（1×10³～1×10⁷）→ 涂布于选择培养基 → 30～37℃培养 1～2 d → 计数（选择菌落数 **30～300** 的平板）

#### 计数方法对比

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 稀释涂布平板法 | 一个单菌落≈一个活菌 | 操作简便 | 多个细胞连在一起时低估 |
| 显微镜直接计数 | 用计数板在显微镜下观察 | 快速得到总数 | 死活菌均计入 |

## 第3节 发酵工程及其应用

### 一、发酵工程的基本环节

```
菌种选育 → 扩大培养 → 配制培养基（灭菌）→ 接种 → 发酵 → 分离提纯产物 → 获得产品
```

| 环节 | 要点 |
|------|------|
| 选育菌种 | 自然界筛选、诱变育种、基因工程育种 |
| 扩大培养 | 发酵罐体积大（几十到几百立方米），需先扩培 |
| 培养基灭菌 | 配方经反复试验确定；设备和培养基必须严格灭菌 |
| 发酵（核心）| 实时监测；严格控制温度/pH/溶解氧；杂菌污染可导致产量大降（如杂菌分泌青霉素酶分解青霉素）|
| 分离提纯 | 菌体本身：过滤/沉淀；代谢物：根据性质提取 |

---

### 二、发酵工程在食品工业的应用

**1. 传统发酵产品**：酱油（黑曲霉蛋白酶）、啤酒（大麦+酵母菌）

**啤酒制作流程**：
大麦→发芽（淀粉酶）→碾磨→糖化→蒸煮（灭菌+风味）→发酵（主发酵+后发酵）→过滤/冷却→装罐/瓶→**消毒**→出售

**2. 食品添加剂**：

| 类型 | 举例 |
|------|------|
| 酸度调节剂 | 柠檬酸（黑曲霉发酵）、乳酸 |
| 增味剂 | 味精（谷氨酸棒状杆菌发酵）|
| 着色剂 | β-胡萝卜素，红曲黄色素 |
| 增稠剂 | 黄原胶、β-环状糊精 |
| 防腐剂 | 乳酸链球菌素、溶菌酶 |

**3. 酶制剂**：α-淀粉酶、果胶酶等 **50多种** 用于食品工业，绝大多数通过发酵工程生产。

---

### 三、发酵工程在医药工业的应用

- **抗生素**：青霉素（1940年代深层通气液体发酵技术产业化）、头孢霉素等
- **氨基酸**：谷氨酸（1957年发酵生产成功）
- **激素**：生长激素释放抑制激素（基因工程：羊脑提取→微生物发酵，成本降为原来的1/几百）
- **基因工程药物**：胰岛素、干扰素、重组乙肝疫苗（将抗原基因转入酵母菌发酵生产）

---

### 四、发酵工程在农牧业的应用

| 应用 | 实例 |
|------|------|
| 微生物肥料 | 根瘤菌肥、固氮菌肥（增进土壤肥力）|
| 微生物农药 | 苏云金杆菌（防80+种农林害虫）、白僵菌（玉米螟）、井冈霉素（水稻纹枯病）|
| 微生物饲料 | 单细胞蛋白（细菌蛋白占干重60%～80%）、青贮饲料（加乳酸菌保鲜）|

---

### 五、发酵工程在其他方面的应用

- **能源**：纤维素废料发酵生产燃料乙醇；CO₂资源化利用（电催化+发酵合成葡萄糖/脂肪酸）
- **碳达峰碳中和**
- **极端微生物**：嗜热菌→洗涤剂；嗜盐菌→提高热敏产品产量

# 第二章 细胞工程

## 第1节 植物细胞工程

### 一、植物细胞工程的基本技术

**理论基础——细胞全能性**：细胞经分裂和分化后，仍具有产生完整生物体的潜能。

> 1902年哈伯兰特提出细胞全能性理论但未成功。1958年斯图尔德证明胡萝卜体细胞可分化为胚。

**为什么不是所有细胞都表现全能性？** → 基因选择性表达（芽原基→芽，叶原基→叶）

#### 植物组织培养

**植物组织培养（plant tissue culture）**：将离体的植物器官、组织或细胞，培养在人工培养基上，诱导其形成完整植株的技术。

**外植体**：离体培养的植物器官、组织或细胞。

**脱分化**：高度分化细胞失去特有结构和功能，形成**愈伤组织**（薄壁组织团块）。

**再分化**：愈伤组织重新分化成芽、根等器官。

**关键激素**：

| 激素 | 作用 |
|------|------|
| **生长素** | 促进生长、分裂、分化 |
| **细胞分裂素** | 促进分裂、**芽的分化** |

- 比值**高**（生长素为主）→ 促进**生根**
- 比值**低**（细胞分裂素为主）→ 促进**生芽**

---

### 二、"菊花的组织培养"实验

**方法步骤**：

```
外植体消毒：70%酒精30s → 无菌水洗2-3次 → 5%次氯酸钠30min → 无菌水洗2-3次
    ↓
切割成0.5～1cm小段，用无菌滤纸吸去表面水分
    ↓
接种到诱导愈伤组织培养基（18～22℃，一般不需光照）
    ↓
愈伤组织形成（约15～20d）
    ↓
转接诱导生芽培养基（有光照）→ 长出芽
    ↓
转接诱导生根培养基 → 生根
    ↓
炼苗（打开封口膜适应自然环境）
    ↓
移栽：洗去根部培养基 → 蛭石/珍珠岩 → 逐步移栽入土
```

**注意事项**：培养基和器械全部灭菌；接种在酒精灯火焰旁；外植体不要倒插；定期观察记录。

---

### 三、植物体细胞杂交技术

**背景**：番茄和马铃薯——希望培育"地上结番茄、地下长马铃薯"的"超级作物"，但杂种植株**未能**实现，原因是基因表达调控问题。

**原生质体**：用**纤维素酶和果胶酶**去除细胞壁后获得的裸露细胞。

**植物体细胞杂交（plant somatic hybridization）**：将不同来源的植物体细胞融合成杂种细胞，培育成新植物体的技术。

**融合方法**：
- 物理法：电融合法、离心法
- 化学法：**PEG融合法**、高Ca²⁺—高pH融合法

**流程**：
A植物细胞 → 去壁 → A原生质体
B植物细胞 → 去壁 → B原生质体
↓ 原生质体融合（PEG/电融合）
↓ 杂种细胞（再生细胞壁）
↓ 愈伤组织
↓ 杂种植株

**已成功案例**：白菜—甘蓝、普通小麦—长穗偃麦草、多种柑橘属间杂种植株（我国）

---

### 四、植物细胞工程的应用

#### 1. 快速繁殖（微型繁殖技术）

- 高效快速繁殖优良品种
- 保持**遗传特性**
- 应用：观赏植物、经济林木、无性繁殖作物、**濒危植物**
- 实例：荷兰兰花产业（60年代起步）、我国铁皮石斛试管苗规模化生产

#### 2. 作物脱毒

- 问题：无性繁殖作物（马铃薯、草莓、香蕉）病毒逐年积累→减产降质
- 方法：切取**茎尖**（病毒极少或无）→ 组织培养 → **脱毒苗**
- 效果：脱毒马铃薯比未脱毒**增产50%以上**

#### 3. 单倍体育种

```
花药/花粉培养 → 单倍体植株 → 诱导染色体加倍 → 纯合二倍体植株
```

- 优点：**年限短**（当年完成）；隐性性状容易显现
- 我国：1974年世界首例"**单育1号烟草**"；后育成水稻、玉米、油菜、甘蓝、甜椒等新品种

#### 4. 突变体的利用

- 组织培养中细胞受培养条件/诱变因素影响产生突变
- 筛选有用突变→培育新品种
- 实例：抗花叶病毒甘蔗、抗盐碱烟草

#### 5. 细胞产物的工厂化生产

- **次生代谢物**：非植物基本生命活动所必需，是药物、香料、色素等重要来源
- **细胞培养**：在离体条件下培养单个植物细胞或细胞团使其增殖

| 实例 | 说明 |
|------|------|
| 紫草宁 | 世界首例药用植物细胞工程产品（20世纪80年代）|
| 紫杉醇 | 治乳腺癌；传统方法破坏濒危红豆杉；现筛选高产细胞我国领先 |
| 人参皂苷 | 我国实现人参细胞培养生产活性成分 |

## 第2节 动物细胞工程

### 一、动物细胞培养

**动物细胞培养（animal cell culture）**：从动物体取出组织，分散成单个细胞，在适宜培养条件下生长和增殖的技术。

#### 培养条件

| 条件 | 要求 |
|------|------|
| 营养 | 合成培养基 + 血清；糖类、氨基酸、无机盐、维生素 |
| 无菌无毒 | 定期更换培养液清除代谢物 |
| 温度 | 哺乳动物细胞：**（36.5±0.5）℃** |
| pH | **7.2～7.4** |
| 气体 | O₂（必需）+ CO₂（维持pH）；95%空气 + 5% CO₂ |
| 渗透压 | 适宜 |

#### 培养过程

```
取动物组织 → 分散成单个细胞（机械法/胰蛋白酶/胶原蛋白酶）
    ↓
制成细胞悬液 → 放入培养瓶/培养皿
    ↓
原代培养（细胞贴壁；发生**接触抑制**）
    ↓
分瓶：胰蛋白酶处理 → 分散成单个细胞 → 离心收集 → 分瓶
    ↓
传代培养
```

#### 两类生长方式

| 类型 | 特点 | 代表 |
|------|------|------|
| 悬浮培养 | 细胞漂浮在培养液中 | 血细胞 |
| 贴壁培养 | 需贴附基质表面；发生**接触抑制** | 大多数细胞 |

---

### 二、干细胞培养及其应用

**干细胞（stem cell）**：在一定条件下可以分化成其他类型细胞的细胞。

#### 胚胎干细胞（ES细胞）

| 项目 | 内容 |
|------|------|
| 来源 | **早期胚胎** |
| 分化能力 | **全能性**：可分化为任何类型细胞，形成完整个体 |
| 研究 | 1981年首次分离小鼠ES细胞；已分离兔、牛、猴、人ES细胞 |
| 局限 | 需从胚胎获取，涉及**伦理问题** |

#### 成体干细胞

| 项目 | 内容 |
|------|------|
| 来源 | 成体组织或器官内 |
| 分化能力 | **组织特异性**：只能分化成特定细胞/组织 |
| 实例 | 骨髓**造血干细胞**、神经**神经干细胞**、睾丸**精原干细胞** |

**造血干细胞**：治疗白血病、恶性肿瘤放化疗后的造血/免疫功能障碍。

#### 诱导多能干细胞（iPS细胞）

| 项目 | 内容 |
|------|------|
| 发现 | 2006年，山中伸弥（日本）|
| 原理 | 体外诱导成纤维细胞（加特定因子）→ 类似胚胎干细胞的细胞 |
| 优点 | 无须破坏胚胎（无伦理争议）；可来源于病人自身体细胞（无免疫排斥）|
| 应用 | 阿尔茨海默病、心血管疾病、镰状细胞贫血等治疗 |

---

### 三、动物细胞融合技术与单克隆抗体

#### 动物细胞融合技术

**原理**：使两个或多个动物细胞结合形成一个细胞，具有原来多个细胞的遗传信息。

**诱导方法**：PEG融合法、电融合法、**灭活病毒诱导法**

> 灭活病毒：病毒表面糖蛋白与细胞膜糖蛋白作用，使细胞膜打开融合。

#### 单克隆抗体

**传统方法问题**：产量低、纯度低、特异性差。

**杂交瘤技术原理**：

```
特定抗原免疫小鼠 → 脾细胞（含多种B淋巴细胞，各分泌不同抗体）
    ↓
与骨髓瘤细胞（无限增殖）融合 → 杂交瘤细胞
    ↓
选择培养基筛选（未融合/同种融合细胞死亡，只有杂交瘤存活）
    ↓
克隆化培养 + 抗体检测 → 筛选出分泌特定抗体的杂交瘤细胞
    ↓
体外大规模培养 or 注射小鼠腹腔增殖
    ↓
单克隆抗体
```

**应用**：
- **诊断试剂**：早孕诊断、疾病鉴定（定位肿瘤/心血管畸形）
- **ADC（抗体—药物偶联物）**：单克隆抗体+细胞毒素，精准杀伤肿瘤
- **新药测试**

> 米尔斯坦和科勒于1984年获**诺贝尔生理学或医学奖**。

---

### 四、动物体细胞核移植技术和克隆动物

#### 体细胞核移植技术

**核移植（nuclear transplantation）**：将一个细胞的细胞核移入去核卵母细胞中，使重新组合的细胞发育成新胚胎，进而发育成动物个体的技术。

| 类型 | 难度 | 说明 |
|------|------|------|
| 胚胎细胞核移植 | 较低 | 分化程度低 |
| **体细胞核移植** | 较高 | 分化程度高，表现全能性困难 |

**克隆猴难点**：非人灵长类体细胞核移植非常困难。
- 供体核不能完全恢复分化前功能状态→胚胎发育率低
- 操作技术尚不完善

#### 流程（以克隆奶牛为例）

```
从屠宰场收集卵巢 → 采集卵母细胞 → 体外培养至MⅡ期
    ↓
从供体高产奶牛取体细胞，培养
    ↓
显微操作法去核（也可用梯度离心/紫外线短照/化学处理）
    ↓
将供体体细胞注入去核卵母细胞
    ↓
电融合法 → 重构胚（具有发育成完整个体的能力）
    ↓
激活：电刺激/Ca²⁺载体/乙醇/蛋白酶合成抑制剂
    ↓
胚胎移植到代孕母牛体内
    ↓
生出与供体遗传物质基本相同的犊牛
```

#### 应用前景

| 领域 | 应用 |
|------|------|
| 畜牧 | 加速遗传改良、促进优良畜群繁育 |
| 医药 | 转基因克隆动物作为**生物反应器**生产药物 |
| 器官移植 | 异种移植；患者自身细胞→避免免疫排斥 |
| 基础研究 | 胚胎发育/衰老研究 |
| 濒危物种 | 增加存活数量 |

#### 存在的问题

- 成功率低；理论基础滞后
- 克隆动物健康问题：体形过大、异常肥胖、发育困难、脏器缺陷、免疫失调
- 多莉早衰问题（存活7年，普通绵羊12岁）

#### "三亲婴儿"

```
母亲卵母细胞（有线粒体遗传病基因）→ 去核
卵母细胞捐献者 → 去核（留健康细胞质）
父亲精子
    ↓
体外受精 + 核移植
    ↓
生出"三亲婴儿"（拥有父亲+母亲+捐献者部分基因）
```

优点：避免母亲线粒体遗传病基因传递（遗传物质主要在细胞核，线粒体基因来自健康捐献者）

## 第3节 胚胎工程

### 一、胚胎工程的理论基础

#### （一）受精

**受精（fertilization）**：精子与卵子结合形成合子（受精卵）的过程。在自然条件下，哺乳动物受精在**输卵管内**完成。

##### 1. 精子获能

刚排出的精子**不能**立即受精，必须在雌性动物生殖道发生相应生理变化后才能获得受精能力——称为**"精子获能"（capacitation）**。

**获能方法**：
- 利用雌性动物生殖道
- 人工配制**获能液**（常用成分：**肝素**、**Ca²⁺载体**）

##### 2. 卵子的准备

- 卵子排出后 **2～3 h** 才能被精子穿入
- 不同动物成熟程度不同：
  - **马、犬等**：初级卵母细胞
  - **猪、羊等**：次级卵母细胞
- 需在输卵管内进一步成熟，到 **MⅡ期** 才具备受精能力

##### 3. 受精阶段（详细7步）

| 步骤 | 内容 |
|------|------|
| 1 | 精子释放酶，溶解卵细胞膜外结构（如透明带）|
| 2 | 精子借助运动接触卵细胞膜 |
| 3 | **精卵细胞膜融合** |
| 4 | **透明带反应**：阻止其他精子进入透明带 |
| 5 | **卵细胞膜反应**：拒绝其他精子进入卵内 |
| 6 | 精子入卵：尾部脱离，核膜破裂 → 形成**雄原核**；卵子完成**减数分裂Ⅱ**，释放第二极体 → 形成**雌原核** |
| 7 | 雄、雌原核充分发育后相向移动，核膜消失 → 形成含**两个染色体组**的合子 = **受精卵** |

> **受精标志**：观察到两个极体；或观察到雄原核+雌原核。

---

#### （二）胚胎早期发育

**卵裂**：受精卵在输卵管内进行有丝分裂，但胚胎**总体积不增加**（细胞数量↑，细胞变小）。

**阶段**：

| 阶段 | 特征 |
|------|------|
| 受精卵 | 分裂开始 |
| 2细胞 → 4细胞 → 8细胞 → 16细胞 | 继续卵裂 |
| **桑葚胚** | 形成致密细胞团，形似桑葚 |
| **囊胚** | 内细胞团（将来发育成胎儿）形成；滋养层细胞（将来发育成胎膜和胎盘）；出现**囊胚腔** |
| **孵化** | 透明带破裂胚胎伸出（不能正常孵化则无法继续发育）|
| **原肠胚** | 外胚层、内胚层、中胚层形成 → 分化成各种组织器官 |

**不同动物受精卵进入子宫时间对比**：

| 动物 | 2细胞 | 4细胞 | 8细胞 | 16细胞 | 桑葚胚 | 进入子宫 |
|------|-------|-------|-------|--------|--------|----------|
| 小鼠 | 24-38h | 38-50h | 50-60h | 60-70h | 68-80h | 3d（桑葚胚）|
| 绵羊 | 36-38h | 42h | 48h | 67-72h | 96h | 3-4d（16细胞）|
| 猪 | 21-51h | 51-66h | 66-72h | 90-110h | 110-114h | 2-2.5d（4-6细胞）|
| 马 | 24h | 30-36h | 50-60h | 72h | 98-106h | 6d（囊胚）|
| 牛 | 27-42h | 44-65h | 46-90h | 96-120h | 120-144h | 4-5d（8-16细胞）|

---

### 二、胚胎工程技术及其应用

#### （一）体外受精

**体外受精（IVF）**：人工操作使卵子在体外受精，经培养发育为早期胚胎后，再进行移植。

**代表成果**：试管牛、试管羊

**步骤**：
```
采集卵母细胞（卵巢，MⅡ期）→ 体外成熟培养
↓
获取精子（新鲜/解冻，离心处理）
↓
精子体外获能
↓
获能精子 + 成熟卵母细胞 → 体外受精（共同培养）
↓
体外培养早期胚胎
↓
胚胎移植
```

我国20世纪80年代起步，人和牛体外受精技术达**国际先进水平**。

---

#### （二）胚胎移植

**胚胎移植（embryo transfer）**：将胚胎移植到同种的、**生理状态相同**的雌性动物体内，使之继续发育为新个体的技术。

| 角色 | 定义 |
|------|------|
| **供体** | 提供胚胎的个体（遗传性状优良）|
| **受体** | 接受胚胎的个体（健康、正常繁殖能力）|

**实质**：早期胚胎在**相同生理环境条件**下进行**空间位置的转移**。

**超数排卵**：应用外源**促性腺激素**，诱发卵巢排出比自然情况下更多的成熟卵子。

**优势**：
- 供体专心生产胚胎，繁殖周期缩短，后代数量是自然繁殖的**十几倍到几十倍**
- 受体承担妊娠和育仔

**效率（牛）**：每次处理一头供体奶牛→产下 **3～4头** 犊牛；每年处理4～5次；一头良种母牛一年可产 **十几头** 后代。

---

#### （三）胚胎分割

**胚胎分割（embryo splitting）**：采用机械方法将早期胚胎切割成2、4、8等份，获得**同卵双胎或多胎**的技术。属于**无性繁殖/克隆**。

**设备**：体视显微镜 + 显微操作仪

**操作对象**：发育良好、形态正常的**桑葚胚或囊胚**

**关键**：分割**囊胚**时，须将**内细胞团均等分割**（否则影响发育）。

**效率**：二分胚胎（1/2）效率最高；分割次数越多成活率越低。

**历史**：1978年小鼠桑葚胚一分为二成功；1979年绵羊胚胎分割获同卵羔羊；我国在小鼠、兔、绵羊、山羊、牛等均成功。

**应用**：促进优良品种繁殖；遗传学研究材料；胚胎移植前**性别鉴定**和**遗传病筛查**。

# 第三章 基因工程

## 第1节 重组DNA技术的基本工具

### 一、限制酶——"分子手术刀"

**限制性内切核酸酶（restriction endonuclease）**：从原核生物中分离，能识别双链DNA分子的特定核苷酸序列，在特定部位断开磷酸二酯键。

- 识别序列：多数由 **6个** 核苷酸组成（如EcoRⅠ、SmaⅠ）
- 部分由4个、8个或其他数量核苷酸组成

**切割产生的末端**：

| 类型 | 切割方式 | 产生 |
|------|----------|------|
| **黏性末端** | 在识别序列中心轴线**两侧**切割 | 两条链交错切开，产生互补黏性末端 |
| **平末端** | 在识别序列**中心轴线处**切割 | 两条链平齐切开 |

---

### 二、DNA连接酶——"分子缝合针"

| 类型 | 来源 | 特点 |
|------|------|------|
| E.coli DNA连接酶 | 大肠杆菌 | 连接黏性末端效率高；连接平末端效率低 |
| T4 DNA连接酶 | T4噬菌体 | 连接平末端效率高 |

**功能**：恢复被限制酶切开的**磷酸二酯键**。

---

### 三、基因工程载体——"分子运输车"

**质粒（plasmid）**：裸露的、结构简单的、独立于拟核DNA之外的环状双链DNA分子，具有**自我复制能力**。

**质粒作为载体的条件**：
- 有限制酶切割位点（供外源DNA插入）
- 有标记基因（四环素抗性基因、氨苄青霉素抗性基因等）→ 便于筛选
- 在天然质粒基础上经过**人工改造**

**其他载体**：噬菌体、动植物病毒等。

---

## 第2节 基因工程的基本操作程序

### 一、基因工程四步流程

```
① 目的基因的筛选与获取
② 基因表达载体的构建
③ 将目的基因导入受体细胞
④ 目的基因的检测与鉴定
```

---

### 二、目的基因的获取

**目的基因**：用于改变受体细胞性状或获得预期表达产物的基因，主要是**编码蛋白质的基因**。

**获取方法**：
- 人工合成
- **PCR扩增**（最常用）：根据DNA半保留复制原理，体外大量扩增目的基因

**PCR原理与条件**：

| 步骤 | 温度 | 过程 |
|------|------|------|
| 变性 | 90℃+ | 双链DNA解聚为单链 |
| 复性 | ~50℃ | 引物与单链DNA相应序列结合 |
| 延伸 | ~72℃ | 耐高温DNA聚合酶（**Taq酶**）催化合成DNA子链 |

> Taq酶来源：1976年中国科学家**钱嘉韵**从嗜热菌中分离（穆里斯PCR诺贝尔奖背后的中国贡献）。

---

### 三、基因表达载体的构建

**组成**：目的基因 + 启动子 + 终止子 + 标记基因 + 复制原点

| 组件 | 作用 |
|------|------|
| **启动子** | RNA聚合酶识别结合部位，驱动基因转录 |
| **终止子** | 使转录在所需位置停止（红色信号灯）|
| **标记基因** | 筛选重组DNA分子 |
| **复制原点** | 使质粒在受体细胞中自我复制 |

---

### 四、将目的基因导入受体细胞

| 受体 | 方法 |
|------|------|
| 植物细胞 | **农杆菌转化转化法**（双子叶植物/裸子植物）；**花粉管通道法**（我国科学家独创）；基因枪法
| 动物受精卵 | **显微注射法** |
| 微生物 | Ca²⁺处理→处于能吸收DNA的生理状态→导入重组质粒 |

**农杆菌转化法**：
- 农杆菌含**Ti质粒**，侵染植物后T-DNA（含目的基因）转移到植物染色体DNA上
- 目前对单子叶植物（如水稻、玉米）也取得成功

---

### 五、目的基因的检测与鉴定

| 水平 | 方法 |
|------|------|
| **分子水平** | PCR检测DNA是否插入；检测是否转录出mRNA；抗原—抗体杂交检测是否翻译出蛋白质 |
| **个体生物学水平** | 饲喂实验检测抗虫性等性状 |

---

## 第3节 基因工程的应用

### 一、在农牧业方面的应用

#### 1. 转基因抗虫植物

- 将Bt抗虫蛋白基因（来自苏云金杆菌）导入棉花→产生Bt抗虫蛋白
- Bt抗虫蛋白在昆虫肠道碱性环境中才有毒性，对人畜无害（胃液呈酸性，无特异性受体）
- 已培育：转基因抗虫棉花、玉米、大豆、水稻、马铃薯

#### 2. 转基因抗病/抗除草剂植物

- 抗病毒甜椒、番木瓜、烟草（转入抗病基因）
- 抗除草剂玉米、大豆、油菜、甜菜（转入降解/抵抗除草剂的基因）

#### 3. 改良植物品质

- 提高某种必需氨基酸含量（如转基因高赖氨酸玉米）
- 改变花色（如转基因矮牵牛）

#### 4. 提高动物生长速率

- 将生长激素基因导入动物（如转基因鲤鱼生长速率提高42%～115%）

#### 5. 改善畜产品品质

- 将肠乳糖酶基因导入奶牛→乳糖不耐受者也能喝牛奶

## 第4节 蛋白质工程的原理和应用

### 一、蛋白质工程

**蛋白质工程**：基于蛋白质结构与功能关系的认识，利用基因工程手段，或直接改造蛋白质，生产出符合人类需要的蛋白质的综合技术。

**与基因工程的区别**：
- 基因工程：基因→蛋白质（"从基因到产品"）
- 蛋白质工程：蛋白质的结构→基因→蛋白质（"逆推设计"）

# 附录：核心概念速记

## 发酵工程

```
发酵 = 微生物在适宜条件下将原料转化为产物
传统发酵技术 = 利用天然菌种，家庭/作坊式，混合菌种固体发酵
发酵工程 = 规模化，单一菌种，严格控制，工业化

培养基 = 水 + 碳源 + 氮源 + 无机盐 + 特殊营养
消毒 = 杀死一部分微生物（不灭芽孢）
灭菌 = 杀死所有微生物（包括芽孢、孢子）
纯培养 = 获得单一菌种的过程
菌落 = 单菌落 ≈ 单个活菌
选择培养基 = 只让目的菌生长（提供特定营养/条件）
计数：30~300/平板
```

## 细胞工程

```
细胞全能性 = 分化细胞仍保留全套遗传物质
脱分化 = 分化细胞 → 愈伤组织（未分化）
再分化 = 愈伤组织 → 芽/根

植物组织培养 = 离体器官/组织/细胞 → 完整植株
原生质体 = 去壁后的裸露细胞
体细胞杂交 = 原生质体融合 → 杂种植株

动物细胞培养 = 离体细胞培养（分散 + 适宜条件）
接触抑制 = 贴壁细胞长满后停止分裂
干细胞 = 可分化成其他类型细胞的细胞
ES细胞 = 胚胎干细胞（全能性）
iPS细胞 = 诱导多能干细胞（体细胞诱导，无伦理争议）

细胞融合 = 两个/多个细胞 → 一个杂交细胞
杂交瘤细胞 = B淋巴细胞（抗体）+ 骨髓瘤细胞（无限增殖）
单克隆抗体 = 单个杂交瘤细胞产生的高度特异性抗体
```

## 胚胎工程

```
受精 = 精子 + 卵子 → 合子
精子获能 = 在雌性生殖道/获能液中获得受精能力
受精标志 = 两个极体 / 雄原核 + 雌原核
透明带反应 = 阻止多精入透明带
卵膜反应 = 阻止多精入卵内

卵裂 = 受精卵有丝分裂，胚胎总体积不增加
桑葚胚 = 致密细胞团（形似桑葚）
囊胚 = 内细胞团 + 滋养层 + 囊胚腔
孵化 = 透明带破裂胚胎伸出

体外受精 = 体外完成受精过程（IVF）
胚胎移植 = 胚胎转移到同种生理状态相同的雌性动物体内
超数排卵 = 用促性腺激素让卵巢排出更多卵子
胚胎分割 = 机械切割早期胚胎 → 同卵双胎/多胎（无性繁殖）

核移植 = 核 + 去核卵母细胞 → 重构胚 → 新个体
克隆动物 = 体细胞核移植产生的个体
重构胚 = 人工重新构建的胚胎
```

## 基因工程

```
限制酶 = "分子手术刀"，识别特定序列切断DNA
DNA连接酶 = "分子缝合针"，恢复磷酸二酯键
质粒 = 环状双链DNA，独立复制，含标记基因和酶切位点
启动子 = RNA聚合酶结合位点，驱动转录
终止子 = 使转录停止

基因工程四步 = 目的基因获取 → 表达载体构建 → 导入受体 → 检测鉴定
PCR = 聚合酶链式反应，体外大量扩增DNA
Taq酶 = 耐高温DNA聚合酶（钱嘉韵1976年从嗜热菌分离）

Bt抗虫蛋白 = 苏云金杆菌产生，昆虫肠道碱性环境下毒性
乳腺生物反应器 = 药用蛋白基因在乳腺中特异表达并分泌
```

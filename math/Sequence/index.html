    高中数学——数列¶

    一、数列基础概念¶

    1.1 数列的定义¶

    数列：按一定次序排列的一列数，记作 \(\{a_n\}\)

    • 通项：第 \(n\) 项 \(a_n\)
    • 前 \(n\) 项和：\(S_n = a_1 + a_2 + \cdots + a_n\)

    1.2 \(a_n\) 与 \(S_n\) 的关系¶

    \[\boxed{a_n = \begin{cases} S_1, & n=1 \\ S_n - S_{n-1}, & n \geq 2 \end{cases}}\]

    注意：已知 \(S_n\) 求 \(a_n\)，必须验证 \(n=1\) 的情况！

    二、等差数列¶

    2.1 定义¶

    若数列 \(\{a_n\}\) 满足 \(a_{n+1} - a_n = d\)（常数），则称 \(\{a_n\}\) 为等差数列，\(d\) 为公差。

    2.2 通项公式¶

    \[a_n = a_1 + (n-1)d\]

    推导：\(a_n = a_1 + (n-1)d\)

    2.3 等差中项¶

    若 \(a, b, c\) 成等差数列，则 \(b = \dfrac{a+c}{2}\)，称 \(b\) 为 \(a\) 与 \(c\) 的等差中项。

    2.4 前 \(n\) 项和公式¶

    \[S_n = \frac{n(a_1 + a_n)}{2} = \frac{n[2a_1 + (n-1)d]}{2}\]

    推导方法——倒序相加法： $\(S_n = a_1 + a_2 + \cdots + a_n\)$ $\(S_n = a_n + a_{n-1} + \cdots + a_1\)$ 两式相加：\(2S_n = n(a_1 + a_n)\)

    2.5 等差数列的常用性质¶

    性质 公式
    对称性 \(a_1 + a_n = a_2 + a_{n-1} = \cdots\)
    下标和 若 \(m + n = p + q\)，则 \(a_m + a_n = a_p + a_q\)
    中项推广 \(a_k + a_{m+n-k} = a_m + a_n\)
    单调性 \(d > 0\) 时单调递增，\(d < 0\) 时单调递减

    2.6 等差数列的判定方法¶

    数列 \(\{a_n\}\) 是等差数列的充要条件： $\(2a_n = a_{n-1} + a_{n+1} \quad (n \geq 2)\)$

    或 $\(a_{n} - a_{n-1} = d \text{（常数）} \quad (n \geq 2)\)$

    三、等比数列¶

    3.1 定义¶

    若数列 \(\{a_n\}\) 满足 \(\dfrac{a_{n+1}}{a_n} = q\)（常数，\(q \neq 0\)），则称 \(\{a_n\}\) 为等比数列，\(q\) 为公比。

    3.2 通项公式¶

    \[a_n = a_1 \cdot q^{n-1}\]

    推导：\(a_n = a_1 q^{n-1}\)

    3.3 等比中项¶

    若 \(a, b, c\) 成等比数列，则 \(b^2 = ac\)，称 \(b\) 为 \(a\) 与 \(c\) 的等比中项。

    3.4 前 \(n\) 项和公式¶

    \[S_n = \begin{cases} n a_1, & q = 1 \\ \dfrac{a_1(1-q^n)}{1-q}, & q \neq 1 \end{cases}\]

    推导方法——错位相减法： $\(S_n = a_1 + a_1q + a_1q^2 + \cdots + a_1q^{n-1}\)$ $\(qS_n = a_1q + a_1q^2 + \cdots + a_1q^n\)$ 两式相减：\((1-q)S_n = a_1(1-q^n)\)

    3.5 等比数列的常用性质¶

    性质 公式
    对称性 \(a_1 \cdot a_n = a_2 \cdot a_{n-1} = \cdots\)
    下标积 若 \(m + n = p + q\)，则 \(a_m \cdot a_n = a_p \cdot a_q\)
    非零性 \(a_n \neq 0\)（等比数列各项均不为零）

    3.6 等比数列的判定方法¶

    数列 \(\{a_n\}\)（\(a_n \neq 0\)）是等比数列的充要条件： $\(a_n^2 = a_{n-1} \cdot a_{n+1} \quad (n \geq 2)\)$

    四、求数列通项公式的方法¶

    4.1 累加法¶

    适用：形如 \(a_{n+1} = a_n + f(n)\) 的递推关系

    \[a_n = a_1 + \sum_{k=1}^{n-1} f(k)\]

    例：\(a_{n+1} = a_n + 2^n,\ a_1 = 1\)

    \[a_n = 1 + \sum_{k=1}^{n-1} 2^k = 1 + (2^n - 2) = 2^n - 1\]

    4.2 累乘法¶

    适用：形如 \(\dfrac{a_{n+1}}{a_n} = f(n)\) 的递推关系

    \[a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k)\]

    4.3 构造等比数列¶

    适用：形如 \(a_{n+1} = p a_n + q\) 的递推关系（\(p \neq 1\)）

    方法：两边加一个常数，构造等比数列

    设 \(a_{n+1} + \lambda = p(a_n + \lambda)\)，解得 \(\lambda = \dfrac{q}{p-1}\)

    则 \(\{a_n + \lambda\}\) 是首项为 \(a_1 + \lambda\)、公比为 \(p\) 的等比数列。

    例：\(a_{n+1} = 2a_n + 1,\ a_1 = 1\)

    解：\(\lambda = \dfrac{1}{2-1} = 1\)

    构造 \((a_{n+1} + 1) = 2(a_n + 1)\)，令 \(b_n = a_n + 1\)

    \(b_n = b_1 \cdot 2^{n-1} = 2 \cdot 2^{n-1} = 2^n\)

    所以 \(a_n = 2^n - 1\)

    4.4 取倒数法¶

    适用：形如 \(\dfrac{1}{a_{n+1}} = \dfrac{1}{a_n} + d\) 的递推关系

    令 \(b_n = \dfrac{1}{a_n}\)，转化为等差数列。

    4.5 取对数法¶

    适用：形如 \(a_{n+1} = p a_n^q\) 的递推关系（\(p > 0\)）

    令 \(b_n = \ln a_n\)，转化为 \(b_{n+1} = \ln p + q \cdot b_n\)。

    4.6 不动点法（进阶）¶

    适用：形如 \(a_{n+1} = \dfrac{p a_n + q}{r a_n + s}\) 的分式线性递推

    设 \(x = \dfrac{p x + q}{r x + s}\)，解得不动点 \(x_1, x_2\)。

    令 \(b_n = \dfrac{a_n - x_1}{a_n - x_2}\)，则 \(\{b_n\}\) 为等比数列。

    4.7 特征根法（进阶）¶

    适用：二阶线性递推 \(a_{n+2} = p a_{n+1} + q a_n\)

    特征方程：\(x^2 = p x + q\)，解得特征根 \(x_1, x_2\)：

    • 若 \(x_1 \neq x_2\)：\(a_n = A x_1^n + B x_2^n\)
    • 若 \(x_1 = x_2\)：\(a_n = (A + Bn) x_1^n\)

    其中 \(A, B\) 由初始条件确定。

    五、求数列前 \(n\) 项和的方法¶

    5.1 公式法¶

    直接使用等差/等比数列求和公式。

    5.2 错位相减法¶

    适用：数列 \(\{c_n\}\) 的通项 \(c_n = a_n \cdot b_n\)，其中 \(\{a_n\}\) 是等差数列，\(\{b_n\}\) 是等比数列。

    步骤：

    1. 写出 \(S_n = a_1 b_1 + a_2 b_2 + \cdots + a_n b_n\)
    2. 两边乘以公比 \(q\)
    3. 两式相减，巧妙错位
    4. 利用错位消去简化求和

    万能公式（等差数列 \(\{a_n\}\) 公差为 \(d\)，等比数列 \(\{b_n\}\) 公比为 \(q\)）： $\(S_n = \frac{a_1 b_1 (1-q^n)}{1-q} - \frac{d b_1 (q - q^n)}{(1-q)^2}\)$

    5.3 裂项相消法¶

    适用：数列通项可拆分为 \(c_n = \dfrac{1}{f(n) - f(n+1)}\) 或类似形式。

    常用裂项公式：

    裂项形式 结果
    \(\dfrac{1}{n(n+1)}\) \(\dfrac{1}{n} - \dfrac{1}{n+1}\)
    \(\dfrac{1}{n(n+k)}\) \(\dfrac{1}{k}\left(\dfrac{1}{n} - \dfrac{1}{n+k}\right)\)
    \(\dfrac{1}{\sqrt{n+1}+\sqrt{n}}\) \(\sqrt{n+1} - \sqrt{n}\)
    \(\dfrac{n}{(n+1)!}\) \(\dfrac{1}{n!} - \dfrac{1}{(n+1)!}\)
    \(\dfrac{2n}{3^n}\) \(A_n\) 型，用错位相减

    关键技巧：裂项后，大部分项相互抵消，只剩首尾两项。

    5.4 分组求和法¶

    适用：数列可拆分为 \(c_n = a_n + b_n\)，其中 \(\{a_n\}\) 和 \(\{b_n\}\) 是等差或等比数列。

    步骤：分组 → 分别求和 → 合并。

    5.5 倒序相加法¶

    适用：推导等差数列求和公式，或数列满足 \(a_i + a_{n-i+1} = C\)（常数）。

    5.6 奇偶并项求和法¶

    适用：数列通项含 \((-1)^n\)，或需要按奇偶项分组求和。

    六、常用求和公式（必须背）¶

    公式 结果
    \(\sum_{k=1}^{n} k\) \(\dfrac{n(n+1)}{2}\)
    \(\sum_{k=1}^{n} k^2\) \(\dfrac{n(n+1)(2n+1)}{6}\)
    \(\sum_{k=1}^{n} k^3\) \(\left[\dfrac{n(n+1)}{2}\right]^2\)
    \(\sum_{k=1}^{n} (2k-1)\) \(n^2\)
    \(\sum_{k=1}^{n} 2k\) \(n(n+1)\)

    七、\(S_n\) 的单调性与最值¶

    7.1 等差数列前 \(n\) 项和的最值¶

    对于等差数列 \(\{a_n\}\)（\(d < 0\)）：

    \[S_n = \frac{d}{2} n^2 + \left(a_1 - \frac{d}{2}\right)n\]

    是关于 \(n\) 的二次函数，开口向下。

    • 当 \(n = -\dfrac{b}{2a} = -\dfrac{2a_1-d}{2d}\) 时，\(S_n\) 取最大值

    7.2 求 \(S_n\) 最大值的方法¶

    1. 二次函数法：将 \(S_n\) 写成 \(An^2 + Bn\) 的形式，求顶点
    2. 临界项法：找 \(a_n > 0\) 且 \(a_{n+1} \leq 0\) 的临界点

    八、常见的递推模型总结¶

    类型 递推形式 求通项方法
    等差型 \(a_{n+1} = a_n + d\) 直接得 \(a_n = a_1 + (n-1)d\)
    等比型 \(a_{n+1} = a_n \cdot q\) 直接得 \(a_n = a_1 q^{n-1}\)
    一次函数型 \(a_{n+1} = p a_n + q\) 构造等比数列
    分式型 \(a_{n+1} = \dfrac{p a_n + q}{r a_n + s}\) 不动点法
    指数型 \(a_{n+1} = p a_n^q\) 取对数法
    阶乘型 \(a_{n+1} = (n+1) a_n\) 累乘法

    九、易错点与注意事项¶

    1. 等比数列求和分类讨论¶

    • 必须先判断 \(q = 1\) 的情况！
    • 当 \(q = 1\) 时，\(S_n = n a_1\)
    • 当 \(q \neq 1\) 时，\(S_n = \dfrac{a_1(1-q^n)}{1-q}\)

    2. \(a_n\) 与 \(S_n\) 关系¶

    • 由 \(S_n\) 求 \(a_n\) 时，必须单独验证 \(n=1\)
    • 很多题目在这里挖坑！

    3. 等比数列各项非零¶

    • 等比数列中 \(a_n \neq 0\)（易忽略！）

    4. 等差数列判定¶

    • 证明等差数列要说明 \(n \geq 2\) 以后 \(a_{n+1} - a_n = d\)
    • 不能仅验证 \(n=1,2,3\) 成立就说是等差数列

    5. 裂项相消注意项数¶

    • 裂项后要检查剩余几项
    • 有时剩首尾两项，有时剩更多

    十、典型高考题型¶

    题型一：等差/等比基本量计算¶

    核心：用方程思想，求 \(a_1, d/q, n\)。

    题型二：判断或证明等差/等比¶

    核心：用定义或等差/等比中项性质。

    题型三：求通项公式¶

    核心：根据递推关系选择合适方法。

    题型四：求前 \(n\) 项和¶

    核心：识别数列类型，选择求和方法。

    题型五：\(S_n\) 与 \(a_n\) 的综合¶

    核心：利用 \(a_n = S_n - S_{n-1}\) 互化。

    题型六：数列与不等式/函数的综合¶

    核心：构造、转化、归纳。

    附录：常见数列前几项观察规律¶

    规律 前 \(n\) 项和 \(S_n\)
    \(1, 2, 3, \cdots, n\) \(\dfrac{n(n+1)}{2}\)
    \(1, 3, 5, \cdots, 2n-1\) \(n^2\)
    \(2, 4, 6, \cdots, 2n\) \(n(n+1)\)
    \(1^2, 2^2, 3^2, \cdots, n^2\) \(\dfrac{n(n+1)(2n+1)}{6}\)
    \(1^3, 2^3, 3^3, \cdots, n^3\) \(\left[\dfrac{n(n+1)}{2}\right]^2\)
